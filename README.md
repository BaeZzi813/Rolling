# 🎈 Rolling (롤링페이퍼 서비스)

> 소중한 사람들에게 따뜻한 마음을 전하는 온라인 롤링페이퍼 플랫폼입니다.

---

## 🚀 Service Link
[👉 Rolling 서비스 바로가기](https://18-2-rolling.netlify.app/)

---

## ✨ 주요 기능 (Key Features)

* **💌 롤링페이퍼 생성**: 누구나 쉽게 메시지 보드를 만들고 공유할 수 있습니다.
* **🎨 커스텀 카드**: 다양한 배경색과 폰트를 선택하여 나만의 메시지 카드를 작성합니다.
* **😍 이모지 반응**: 카드마다 이모지를 남겨 풍성한 소통이 가능합니다.
* **🔗 간편 공유**: 생성된 URL을 복사하여 지인들에게 빠르게 전달할 수 있습니다.
* **📱 반응형 디자인**: PC와 모바일 어디서나 최적화된 화면을 제공합니다.

---

## 🛠 기술 스택 (Tech Stack)

### Frontend
- **Framework**: React (Vite 빌드 도구 활용)
- **Styling**: Tailwind CSS (일관된 디자인 시스템 및 반응형 구현)
- **Code Quality**: ESLint, Prettier

### Infrastructure & Tools
- **Deployment**: Netlify
- **Collaboration**: Git & GitHub (체계적인 버전 관리 및 워크플로우 활용)

---

## 📂 프로젝트 구조 (Architecture)

효율적인 협업을 위해 **공용 컴포넌트 중심의 모듈화 구조**로 설계되었습니다.

* **컴포넌트 설계**: 버튼, 카드, 텍스트 필드, 모달 등 재사용 가능한 UI 단위 개발
* **페이지 구성**: 메인, 롤링페이퍼 리스트, 포스트 작성, 메시지 상세 페이지
* **API 설계**: 효율적인 데이터 연동을 위한 인터페이스 구축

```text
src/
 ├── components/    # 공용 컴포넌트
 ├── pages/         # 페이지 구성 (List, Post, Message 등)
 ├── hooks/         # 커스텀 훅
 ├── utils/         # 유틸리티 함수
 └── assets/        # 이미지 및 정적 파일
```
